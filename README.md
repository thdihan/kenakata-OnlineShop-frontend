# KENAKATA - ONLINE SHOP

"Kenakata" is a ecommarce website project built as frontend of a Java, Spring Boot Based Web Application.

Thirt Party Plugins :
- Bootstrap 5
- AOS - Animate On Scroll Library
- Isotope - Filter & sort magical layouts
